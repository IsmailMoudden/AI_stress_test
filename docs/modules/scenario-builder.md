# Scenario Builder

## Purpose

The Scenario Builder will help users transform an initial stress idea into a structured, reviewable scenario draft.

## Future Responsibilities

- ask clarification questions
- formalize scenario objectives
- capture assumptions
- identify affected markets and risk factors
- retrieve historical analogues
- collect supporting evidence
- distinguish AI suggestions from user-approved inputs

## Non-Responsibilities

- portfolio valuation
- official PnL calculation
- autonomous scenario approval
- direct risk engine execution

## Governance Requirements

Every AI suggestion, user edit, accepted assumption, rejected assumption, and evidence reference should be traceable.
