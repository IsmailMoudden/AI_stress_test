# Roadmap

## Phase 0: Repository Foundation

- Establish repository structure.
- Document architecture philosophy.
- Define coding, contribution, security, and testing standards.
- Create module documentation placeholders.

## Phase 1: Platform Skeleton

- Introduce backend service runtime.
- Introduce frontend application shell.
- Add authentication and RBAC foundations.
- Add database migration setup.
- Add CI checks.

## Phase 2: Scenario Drafting

- Implement scenario draft model.
- Add audit trail for user edits.
- Introduce governed AI suggestion capture.
- Add evidence and assumption workflows.

## Phase 3: Scenario Engineering

- Add risk factor mapping.
- Add explainable shock proposal workflows.
- Add versioning and comparison.

## Phase 4: Risk Engine Integration

- Integrate with approved pricing and risk infrastructure.
- Track execution requests and official outputs.
- Add reporting workflows.
